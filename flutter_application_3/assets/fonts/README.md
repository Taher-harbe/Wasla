# تعليمات تنزيل الخطوط

يجب تنزيل ملفات الخطوط التالية ووضعها في هذا المجلد:

## خط Manrope
قم بتنزيل خط Manrope من Google Fonts:
https://fonts.google.com/specimen/Manrope

الملفات المطلوبة:
- Manrope-Regular.ttf
- Manrope-Medium.ttf
- Manrope-SemiBold.ttf
- Manrope-Bold.ttf

## خط Inter
قم بتنزيل خط Inter من Google Fonts:
https://fonts.google.com/specimen/Inter

الملفات المطلوبة:
- Inter-Regular.ttf
- Inter-Medium.ttf
- Inter-SemiBold.ttf
- Inter-Bold.ttf

## خطوات التنزيل:
1. اذهب إلى الرابط أعلاه
2. اضغط على "Download family"
3. استخرج ملف ZIP
4. انسخ ملفات .ttf المطلوبة إلى هذا المجلد (assets/fonts)
5. قم بتشغيل `flutter pub get` في Terminal
